# ck_vip_decoder (2020.11.03 제작)


## 설명
Exploit Kit중 하나인 CK VIP의 최종 유포지 복호화 스크립트입니다.


CK VIP는 최종 유포지의 주소에 대응하는 아스키 코드에 159를 더한 정수 값을 배열에 넣고, 각 요소에 -159연산을 통해 최종 유포지를 복호화한다는 특징이 있습니다.
